function changeCurrentSongDetails(songObj) {

But what does the first line mean?

$('.current-song-image').attr('src','img/' + songObj.image)

1. This means we are using jQuery to select the element with class 'current-song-image'

Then we are setting it's src attribute to something (.attr('src',...)

The src is made of two strings added together: folder name + fileName

This src attribute has a value of img/song1.jpg or whatever songObj.image's value is

What's a CDN?

Content Delivery Network (CDN)

Simply said, a CDN helps us get the files we need for our webpage faster

When you request a file from a CDN, it finds the closest server that has a copy of the file and gives it to you

Why is this useful? Mumbai se Delhi aane mein kam samay lagtaa hai. USA se Delhi aane mein toh bahut zyada lagta hai

Jitna kam samay lagega samaan (fonts/imgs/css) aane mein for the website, the faster the website will look and feel

Add a CSS file for DataTables like this: 

<link rel="stylesheet" type="text/css" href="font-awesome.css">
    <link rel="stylesheet" type="text/css" href="https://cdn.datatables.net/1.10.15/css/jquery.dataTables.min.css">
    <link rel="stylesheet" type="text/css" href="main.css">

<script src="jquery.js">
